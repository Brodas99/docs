## Authentication

All API endpoints are either authenticated using Bearer tokens (JWT) or x-api-key headers. The Bearer token and x-api-key is passed in the Authorization header as a Bearer token.

```json
"security": [
  {
    "cbh-auth.cbh-auth-token": 'value,
    'x-api-key': 'value'
  }
]
```
